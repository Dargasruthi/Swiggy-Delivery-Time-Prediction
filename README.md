# 🚚 Swiggy Delivery Time Prediction

## 📌 Project Overview
This project predicts delivery time using Machine Learning models.

## 🧠 Models Used
- Linear Regression
- Decision Tree
- Random Forest
- Gradient Boosting
- KNN

## ⚙️ Tech Stack
- Python
- Scikit-learn
- Streamlit
- Docker

## 🌐 Live Demo
👉 https://huggingface.co/spaces/DargaSruthi/Swiggy-Delivery-Time-Prediction

## 📊 Features
- Predict delivery time based on:
  - Distance
  - Traffic
  - Weather
  - Ratings
  - Order details

## 🚀 Deployment
Deployed using Streamlit on Hugging Face Spaces.
